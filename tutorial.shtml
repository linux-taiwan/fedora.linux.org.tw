基本

新手一定要會的就是怎麼關掉 SELinux 啊！

關於使用 GNOME 桌面環境。

多媒體相關功能請安裝 RPMFusion 套件庫。

播放 DVD 的話，請再安裝 libdvdcss。

開啟 RAR 壓縮檔可以直接安裝 unar
sudo yum install -y unar
遇到有問題的檔案再安裝 RPMFusion 裡的 unrar 來處理。

雖然有預載 LibreOffice ，但是目前需要自己手動安裝中文語系：
sudo yum install -y libreoffice-langpack-zh-Hant
